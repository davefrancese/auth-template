# Auth Template

This Auth Template uses the following technologies

- [x] Node / Express
- [x] MongoDB
- Passport.js
- [ ] Passport Google OAuth
- [ ] Passport Facebook OAuth
- [ ] Passport Twitter OAuth
- [ ] Passport LinkedIn OAuth
- [ ] Passport Slack OAuth
- [ ] Passport Local (Username/Password)

To begin:

```
$ npm install
```

Entry is index.js
